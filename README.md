# 🎮 Pac-Man Clone

A simple yet fun Pac-Man clone built with **vanilla JavaScript**, **HTML**, and **CSS**. Relive the classic arcade experience in your browser!

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://eyukaz.github.io/pacman)
![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)

---

## 🚀 Features

- **Classic Pac-Man Movement**: Navigate the maze using arrow keys.
- **Ghost AI**: Four colorful ghosts chase Pac-Man with random movement.
- **Dot Collection**: Collect dots to increase your score.
- **Wall Collisions**: Pac-Man and ghosts cannot pass through walls.
- **Game Over**: Get caught by a ghost, and it's game over!
- **Restart Functionality**: Play again with the click of a button.

---

## 🕹️ How to Play

1. **Start the Game**: Click the "Start Game" button.
2. **Controls**:
   - **Arrow Keys** to move Pac-Man:
     - **↑** Up
     - **↓** Down
     - **←** Left
     - **→** Right
3. **Objective**:
   - Collect all the white dots to increase your score.
   - Avoid the ghosts! If a ghost catches you, it's game over.
4. **Restart**: Click "Start Game" again to reset and play again.

## 🐛 Troubleshooting

- **Pac-Man isn't moving**:
  - Ensure you've clicked "Start Game."
  - Make sure your keyboard focus is on the game window (click the canvas if needed).
- **Ghosts aren't moving**: Refresh the page and start a new game.
- **Dots aren't appearing**: Ensure the maze array in `script.js` is correctly defined.
